# mi-primera-app-streamlit


una base de una "red social" de una biblioteca.

el sistema de archivos (base de datos) tiene que ver con la carpeta "data", en esta cuando se crea una cuenta, en el usuarios.db está el nombre y correo con el id único de cada cuenta

en la carpeta pages están todas las paginas que no son del inicio para que streamlit las coloque como paginas separadas

en la carpeta imagenes, están todas las imagenes que se utilizen en el proyecto.

y en el apartado scripts están por ahora los scripts para solicitar datos, por ejemplo está el script para saber la info de los usuarios en usuarios.db ya que sino no se pueden leer porque están encriptados.
